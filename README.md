# MezoryEngine
Simple open source engine
 ##  ***_CMake_***
  **1. cd "path to MezoryEngine", for example: cd "D:\MezoryEngine"**
  
  **2. mkdir build && cd build && cmake ..**
  
  **3. Look in build folder which will contain project files**
  
  # Tested on: Windows
